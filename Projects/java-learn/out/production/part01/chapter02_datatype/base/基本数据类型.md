+ 基本数据类型
    + 布尔类型
    + 字符型
    + 整型
    + 实型
    
+ 布尔类型（Boolean）
    + 只有 true 和 false 两种值，默认值为 false。
    
+ 字符型（char）
  + 采用 Unicode 编码
  + 2 字节
  
+ 整型
  + byte —— 字节类型，1 字节
  + short —— 短整型，2 字节
  + int —— 整型，4 字节
  + long —— 长整型，8 字节
  
+ 实型
  + float —— 单精度浮点型，4 字节
  + double —— 双精度浮点型，8 字节
